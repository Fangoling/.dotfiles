# .dotfiles

## Description
My Dotfiles. (WIP)

## Setup
Run
` git clone "https://github.com/Fangoling/.dotfiles ~/.dotfiles" && ~/.dotfiles/script/setup.sh`
This setups up symbolic links to all files of this repo.
